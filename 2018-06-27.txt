 Вот небольшой список слов, который может встретиться при разговоре или чтении статей в Интернете.
application (также software application) — программное приложение. Компьютерная программа, разработанная для выполнения определенных задач.
background — предыстория. События в прошлом человека или компании. Часто      используется, когда говорят о профессиональном опыте человека.
database — база данных. Способ организации больших объемов информации.
employ — нанимать, брать кого-либо на работу.
implement — реализовывать.
to install software — устанавливать программное обеспечение.
integrate — интегрировать. Объединить два или более в одно целое.
outsource — осуществлять аутсорсинг. Пользоваться услугами сторонних компаний для реализации собственных проектов.
solve — решать.
specifications — спецификация. Детальное описание (продукта).
task — задача.
tool — инструмент.
transfer — переводить, перемещать, пересылать.
webinar — вебинар. Интернет-семинар.
